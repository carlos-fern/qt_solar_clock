# qt_solar_clock
## Description
A Simple solar cock based on Qt5 OpenGLWidgets

## Features
- Zoom
- Tilt
- Sim speed control

## Build & Run

### Script (QT5 Provided)
- sudo chmod +x builder.sh
- sudo ./builder.sh
- ./qt_solar_clock

### Cmake (QT5 Assumed)
- mkdir ./build
- cd ./build
- cmake ..
- make 
- cd ./.
- ./qt_solar_clock

## Dependencies
- QT5
- CMake >= 2.8.11
