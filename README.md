# qt_solar_clock
## Description
A Simple solar cock based on Qt5 OpenGLWidgets. Uses [NASA reference data](https://omniweb.gsfc.nasa.gov/coho/helios/planet.html)

## Features
- Zoom
- Tilt
- Sim speed control

## Build & Run

### Cmake (QT5.5 Provided)
- sudo chmod +x solar_launcher.sh
- mkdir ./build
- cd ./build
- cmake ..
- make 
- (Go for a walk, 30min+ build-time)
- cd ../
- ./solar_launcher.sh

### Script (QT5 Provided)
- sudo chmod +x builder.sh
- sudo ./builder.sh

## Dependencies
- Ubuntu 16.04
- g++
- build-essential
- pkg-config
- CMake >= 2.8.11

## System Requirements
- 4GB+ Free space
- GPU (Recommended)
